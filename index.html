Quick start

How Ending Tracker works

1. Add your book

Start with Add book. Enter the title, then add the total path count and ending count if your book provides them. Ending numbers are created automatically from the total ending count, and each book keeps its own paths, endings, progress, and map.

2. Create a path

Open a book and choose New path. A path is one route through the story. Use In progress for an active route, Draft for a saved branch, Completed for a finished route without a numbered ending, Reached ending when it links to an ending, and Abandoned for routes you stopped tracking.

3. Add page and decision steps

Inside a path, use Continue to add steps in reading order. Add a Page step when you arrive on a page. Add a Decision step when the book asks you to choose, then enter the page number and the selected choice you followed.

4. Create draft branches automatically

For a decision, enter every available option in All available options, one option per line. Keep Create draft paths for unchosen options turned on. When you save the decision, the selected choice stays in the current path and the other choices become Draft paths that inherit the same history up to that decision.

5. Resume from any saved branch

Draft paths appear in the path list. Open one to continue from the exact decision point where it was created, instead of starting over from page 1. Pin a path when you want it to stay easy to find as your current route.

6. Review and reorder history

The History tab shows every saved step. Use Edit or Delete on a step if something was entered incorrectly. Drag and drop steps in History to reorder them when needed; linked draft branches are updated so their inherited history stays aligned.

7. Mark endings reached

When a path reaches a numbered ending, choose the ending in the path details and save. You can also tap an ending from the Endings grid to link it to existing paths, mark it reached, or mark it unreached again.

8. Track progress

The dashboard shows paths discovered, endings reached, completion percentage, active paths, draft branches, and finished paths. Counts update as you add paths, save drafts, complete routes, and link endings.

9. Find similar paths

Open a path and check Similar. The app compares early steps and shows other routes with matching page and decision history, which helps you avoid repeating the same route by accident.

10. Visualize the path map

Use Visualize Map from the book dashboard to see active routes as a branching map. Filter by path or reached ending, then open a highlighted path when you want to inspect or continue it.

11. Back up your progress

Your data autosaves in this browser with localStorage. Use Export Data to download a backup JSON file, and Import Data to restore or move your tracker to another browser.
